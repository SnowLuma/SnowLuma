# 贡献者公约 (Contributor Covenant Code of Conduct)

本文档改编自 [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)，是 SnowLuma 社区共同遵守的行为准则。

## 我们的承诺

身为社区成员、贡献者、维护者，我们承诺：让每一个参与 SnowLuma 项目和社区的人，不因年龄、体型、可见或不可见的身体差异、身心障碍、族裔、性别特征、性别认同与表达、经验水平、教育背景、社会经济地位、国籍、外貌、种族、宗教信仰、或性取向，而遭受骚扰。

我们承诺以营造开放、热情、多元、包容、健康社区的方式行事和互动。

## 我们的准则

有助于建立正向社区环境的行为包括但不限于：

- 对他人展现同理心与善意
- 尊重不同的意见、观点与经验
- 给予并优雅地接受建设性反馈
- 承担责任并向受我们错误影响的人致歉，并从中吸取教训
- 关注的不仅是个人，更是整个社区的最大利益

不可接受的行为包括但不限于：

- 使用情色化的语言或意象，及任何形式的性关注或性骚扰
- 嘲讽、侮辱或贬损性的评论，以及人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可，公布他人的私人信息，例如物理或电子邮件地址
- 其他在专业场合中可被合理认定为不当的行为

## 适用范围

本行为准则适用于所有社区场所，也适用于个人在公开场合代表社区时的言行。代表本社区的情形包括但不限于：使用项目官方电子邮件地址、通过项目官方社交媒体账号发文、以指定代表身份出席线上或线下活动。

具体到 SnowLuma，本准则适用于：

- 本仓库（含 issue、PR、discussion、commit message、code review 评论）
- 项目相关的 QQ 群、Telegram 群以及未来可能新增的官方社群
- 任何以 SnowLuma 名义对外发起的活动或交流

## 执行责任

社区维护者负责诠释并执行我们所认定的可接受行为标准，并将对任何他们认为不当、具威胁性、冒犯性或有害的行为采取适当且公正的纠正措施。

社区维护者有权且有责任删除、编辑或拒绝任何不符合本行为准则的评论、提交、代码、wiki 编辑、issue 及其他贡献，并将在适当时机说明做出审核决策的理由。

## 举报方式

如果你目击或亲身经历了不当行为，请通过以下任一渠道**私下**联系维护者：

- QQ：[SnowLuma-QQ](https://qm.qq.com/q/g3UMLpWALe)（私聊管理员，请勿在群内公开）
- Telegram：[SnowLuma-TG](https://t.me/napcatqq)（私聊管理员）

请勿在 GitHub issue 或公开频道直接发布举报内容，以免对相关方造成二次伤害。

所有投诉都会被及时和公平地审查与调查。所有社区维护者都有义务尊重举报者的隐私与人身安全。

## 执行准则

社区维护者将依据以下「社区影响指引」来判定违反本行为准则之行为的处置方式：

### 1. 更正 (Correction)

**社区影响**：使用了不恰当的言语，或其他在社区中被视为不专业或不受欢迎的行为。

**后果**：由社区维护者私下提出书面警告，明确指出该行为违反规则之处并解释为何不妥。可能要求公开致歉。

### 2. 警告 (Warning)

**社区影响**：通过单一事件或一系列行为造成违规。

**后果**：警告并说明持续行为的后果。在指定时间内禁止与相关人员互动，包括主动联系本行为准则执行者。这包括避免在社区场所及外部渠道（如社交媒体）的互动。违反这些条款可能导致临时或永久封禁。

### 3. 临时封禁 (Temporary Ban)

**社区影响**：严重违反社区准则，包括持续的不当行为。

**后果**：在指定时间内禁止与社区进行任何形式的互动或公开交流。在此期间不得与相关人员有任何公开或私下的互动，包括主动联系本行为准则执行者。违反这些条款可能导致永久封禁。

### 4. 永久封禁 (Permanent Ban)

**社区影响**：表现出违反社区准则的模式，包括持续的不当行为、骚扰个人、或对某类群体进行攻击或贬低。

**后果**：永久禁止在社区内进行任何形式的公开互动。
